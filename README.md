# GROQ-CLI

A simple golang CLI tool using GROQ API and Charm.sh libraries

## Installation

```sh
  go mod tidy
  go run .

```

OR use the binary in the repo

OR `go build .`

## Env variables

- Rename .env.example to rename and configure the values
